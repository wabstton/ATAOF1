Sistema de Gerenciamento de Livros

Uma aplicação para organização e gerenciamento de livros, utilizando algoritmos de ordenação como Bubble Sort e Quick Sort.

Descrição Geral

Este sistema permite que o usuário adicione, liste e organize livros utilizando algoritmos eficientes de ordenação. Ele implementa dois métodos de ordenação: Bubble Sort para títulos e Quick Sort para autores, tornando a aplicação prática e intuitiva. A capacidade máxima do sistema é de 100 livros.

1. Funcionalidades Principais

  • Adicionar Livros: Adiciona um novo livro ao acervo com informações como título, autor e ano de publicação.
  • Listar Livros: Exibe todos os livros cadastrados no sistema.
  • Ordenar Livros: Ordena os livros por título (Bubble Sort) ou por autor (Quick Sort).

2. Interface Gráfica

A interface é desenvolvida com Swing em Java e contém:

  • Formulário de Adição: Campos para inserir informações sobre o livro.
  • Área de Exibição: Um painel que lista todos os livros cadastrados.
  • Botões de Ação: Opções para ordenar a lista de livros por título ou autor.

3. Algoritmos de Ordenação

Ordenação por Título (Bubble Sort):

O algoritmo Bubble Sort compara pares consecutivos de livros e faz trocas sempre que necessário, até que todos estejam ordenados.

Ordenação por Autor (Quick Sort):

O Quick Sort é mais eficiente. Ele seleciona um pivô e ordena a lista recursivamente, separando os livros em sublistas com base em suas comparações com o pivô.

4. Validações e Tratamento de Erros

O sistema valida a entrada do usuário para garantir que o ano de publicação seja um número inteiro. Caso contrário, uma mensagem de erro será exibida. Além disso, o sistema exibe um aviso quando o limite de livros é atingido.

5. Capacidade e Limitações

A aplicação suporta até 100 livros e utiliza o algoritmo Bubble Sort para ordenação por título, que é menos eficiente para listas grandes. Já o Quick Sort, implementado para a ordenação por autor, é mais eficiente e adequado para grandes volumes de dados.

6. Considerações Finais

Esta aplicação fornece uma interface amigável para gerenciamento de livros com funcionalidades práticas de ordenação. Em versões futuras, funcionalidades como remoção e pesquisa de livros podem ser adicionadas para aprimorar a experiência do usuário.
